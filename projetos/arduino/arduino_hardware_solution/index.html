Arduino HardWare Solution

Biblioteca para arduino, destinada a facilitar o controle de circuitos embarcados.

Fundamentos

Esta biblioteca é baseada em Threads, função que não está incluida na biblioteca padrão do Arduino. Por isso é fundamental a instalação da biblioteca ArduinoThread-master.

Outra biblioteca de terceiros utilizada neste projeto, é a PortExpander_I2C, mas não é nescessario baixar-la, a menos que queira utilizar a classe CI01.
